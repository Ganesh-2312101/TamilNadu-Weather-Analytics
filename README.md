#  Tamil Nadu Real-Time Weather Analytics Dashboard

##  Project Overview

This project is an automated weather analytics system that collects real-time weather data for all 38 districts of Tamil Nadu and provides interactive insights through Power BI dashboards.

The system fetches live weather information from the Open-Meteo API, stores the data in a MySQL database, and visualizes key weather metrics such as temperature trends, wind speed analysis, district-wise comparisons, and seasonal patterns.

---

##  Features

- Real-time weather data collection
- Coverage of all 38 Tamil Nadu districts
- Automated data fetching using Python
- MySQL database integration
- Interactive Power BI dashboard
- District-wise temperature monitoring
- Wind speed analysis
- Temperature category distribution
- Trend and seasonality analysis
- Live refresh status tracking
